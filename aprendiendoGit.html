#1 Creando un repositorio en Git

Empiezo escribiendo los comando 'git init' y ya esta!!! lo que hace este comando es crearme una carpeta oculta en el directorio en el que voy a trabajar y automaticamente git ya esta trabanajando el el. Tambien guardo los cambios echos en los archivos escribiendo 'git add .' (enter) 'git commit -m('mensaje')'. git rm --cached: Elimina los archivos del área de Staging y del próximo commit pero los mantiene en nuestro disco duro. git rm --force: Elimina los archivos de Git y del disco duro. 'git reset HEAD': Este es el comando para sacar archivos del área de Staging. No para borrarlos ni nada de eso, solo para que los últimos cambios de estos archivos no se envíen al último commit, a menos que cambiemos de opinión y los incluyamos de nuevo en staging con git add.
